# Elephant

An Elixir STOMP client.

## Status

Not recommended for production use yet.

## Installation

First, add Elephant to your `mix.exs` dependencies:

```elixir
def deps do
  [{:elephant, "~> 0.1.0"}]
end
```

Then, update your dependencies:

```sh-session
$ mix deps.get
```

See https://hexdocs.pm/elephant/Elephant.html
